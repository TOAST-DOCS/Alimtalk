* [Console] ブランドメッセージ動画アップロード機能追加

    * ブランドメッセージ・フレンドトーク広告代替送信時に 080 受信拒否番号がない一部のケースで一般メッセージとして送信されていた問題を修正しました。

### 2026. 05. 27.
* [API] カカオビズセンター統計照会 API 追加
    * カカオビズセンターで提供するお知らせトーク、ブランドメッセージの送信統計およびテンプレート統計を照会できる API が追加されました。
    * 発信キー基準で日別（DAILY）または月別（MONTHLY）統計データを照会できます。

* [API、コンソール] ブランドメッセージ画像削除関連ガイド修正
    * ブランドメッセージテンプレート修正時に画像を変更すると、既存の画像がカカオ CDN から削除され、同一画像を使用する他のテンプレートも影響を受けるという内容でガイドが修正されました。

* [API、コンソール] ブランドメッセージタイプ別内容長さ制限変更
    * IMAGE タイプのメッセージ内容（content）長さ制限が最大 400 文字から最大 1,300 文字に変更されました。
    * IMAGE タイプの改行制限が最大 29 個から最大 99 個に変更されました。
    * WIDE、PREMIUM_VIDEO タイプの改行制限が最大 1 個から最大 5 個に変更されました。

### 2026. 02. 24.
* [API] ブランドメッセージカルーセル固定置換子機能追加
    * カルーセルタイプテンプレートで各アイテムに異なる置換子値を適用できます。
    * path ベースパラメータ（`キー@$.carousel.list[インデックス]`）を通じてアイテム別パーソナライズが可能です。
    * テンプレート置換子キーに「@」文字を使用することはできません。

* [API] ブランドメッセージテンプレート置換子キー有効性検査追加
    * テンプレート登録・修正時に置換子キーに「@」文字が含まれるとエラーが発生します。

* [API] ブランドメッセージ送信時テンプレートパラメータ欠落検査追加
    * 送信要求時にテンプレートで定義された置換子キーが templateParameter に欠落するとエラーが発生します。

* [API、コンソール] ブランドメッセージ BT ボタンタイプ対応
    * ブランドメッセージで BT（ボット転換）ボタンタイプを対応します。
    * コンソールで BT ボタンタイプを選択してテンプレートを登録できます。

* [API、コンソール] ブランドメッセージ AC ボタンタイプ追加
    * ブランドメッセージで AC（チャンネル追加）ボタンタイプを対応します。
    * AC ボタンの名前は必ず「チャンネル追加」である必要があります。
    * カルーセルの場合、全体アイテム中 AC ボタンは 1 個のみ使用可能です。

* [コンソール] お知らせトーク・ブランドメッセージ代替送信レスポンスメッセージ分類改善
    * 代替送信時にレスポンスメッセージが分類されて表示されます。

* [API] テンプレートパラメータ長さ制限追加
    * テンプレートパラメータのキーと値はそれぞれ最大 1,300 文字に制限されます。

* [API] 受信者グルーピングキー長さ制限追加
    * recipientGroupingKey フィールドの最大長が 100 文字に制限されます。

    * SMS サービスアプリキー初期化時に既存に登録した代替送信設定が削除されます。

### 2020. 10. 27.
* [API]お知らせトークPC表示/未表示フィールド変更
    * フィールドがpcFlagからsecurityFlagに変更されました(基本値: false).
    * セキュリティテンプレートかどうかを表すフィールドで、OTP などセキュリティメッセージの場合、設定する必要があります。
    * trueで登録すると、メインデバイスを除くすべてのデバイスにメッセージテキストが表示されません。
* [API]お知らせトークチャンネル追加ボタンタイプを変更
    * チャンネル追加ボタンのタイプがCAからACに変更されました。
    * チャンネル追加ボタンはボタンリスト中、最上段または単独で存在する時のみ登録可能です。
    * チャンネル追加ボタンの名前は「チャンネル追加」のみ可能です。
    * ACに変更したボタンはハイライト処理とアイコンが表示される新しいボタンの形式で表示されます。
* [コンソール]お知らせトーク送信結果バックアップ機能追加
    * お知らせトーク送信結果バックアップリクエスト機能が追加されました。
    * コンソールで照会した検索条件をもとに送信結果Excelファイルを作成できます。
    * 送信結果ファイルは7日が経過すると削除されます。
* [コンソール]カカともへのメッセージ送信結果詳細照会モーダルを改善
    * カカともへのメッセージ送信結果詳細を照会する時、広告かどうかと代替送信リクエスト結果を追加で確認できます。
