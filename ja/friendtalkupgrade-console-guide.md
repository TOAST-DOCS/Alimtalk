## ブランドメッセージ送信

<a id="regular-send"></a>

### 一般送信

発信プロフィールを設定し、内容を入力してブランドメッセージ形式のメッセージを送信できます。
ブランドメッセージを送信するには、コンソールで **Notification > KakaoTalk Bizmessage > ブランドメッセージ**を選択します。

<a id="when-using-a-template"></a>

### テンプレート使用時

![friendtalkupgrade_04_20250616.png](https://static.toastoven.net/prod_alimtalk/friendtalkupgrade/friendtalkupgrade_04_20250729.png)

1. 発信プロフィールを選択します。
    * 発信プロフィールは **Notification > KakaoTalk Bizmessage > 発信プロフィール管理**タブで確認できます。
2. プッシュ通知の可否を選択します。
    * プッシュ通知を**使用しない**に選択した場合、メッセージプッシュ通知が送信されません。
3. テンプレート使用の可否を**使用**に選択した後、送信リクエストするテンプレートを選択します。
    * テンプレートは **Notification > KakaoTalk Bizmessage > ブランドメッセージ > テンプレート管理**タブで登録可能です。
4. 代替送信メッセージ設定の可否を選択します。
    * **080受信拒否管理**タブで代替送信の可否を**使用**に設定し、SMS サービスを連動する必要があります。
5. 080受信拒否番号設定の可否を選択します。
    * 080受信拒否番号設定の可否を**設定しない（発信プロフィール設定で送信）**に選択した場合、発信プロフィールで設定した080受信拒否番号が使用されます。
    * **共通内容で設定**または**ユーザー別設定**を選択した場合、入力した080受信拒否番号が送信に使用されます。
    * **080受信拒否管理**タブで SMS サービスを利用する場合、SMS サービスに登録された080受信拒否番号のみ使用できます。
6. 受信者を追加します。
    * 受信者は携帯電話番号形式で作成できます。
7. 受信者別にターゲティングタイプを選択します。
    * ブランドメッセージは、広告主のマーケティング受信同意（以下「マ受同」）ユーザーおよびチャンネルフレンドユーザーを対象に送信できる広告性メッセージです。
    * ターゲティングタイプによって、指定した受信者のメッセージ受信可否が変わる場合があります。
        * M：広告主マ受同ユーザー（カカオトーク受信同意）
            * 広告主のマ受同ユーザー（カカオトーク受信同意）を対象に広告性メッセージを送信します。
        * N：広告主マ受同ユーザー（カカオトーク受信同意）- チャンネルフレンド
            * 広告主のマ受同ユーザー（カカオトーク受信同意）からチャンネルフレンドを除いて広告性メッセージを送信します。
        * I：広告主送信リクエスト対象 ∩ チャンネルフレンド
            * 広告主送信リクエスト対象で、チャンネルフレンドの場合にのみ広告性メッセージを送信します。
8. 入力を完了した後、**送信**をクリックして送信します。

<a id="when-not-using-a-template"></a>

### テンプレート未使用時

![friendtalkupgrade_05_20250616.png](https://static.toastoven.net/prod_alimtalk/friendtalkupgrade/friendtalkupgrade_05_20250729.png)

1. 発信プロフィールを選択します。
    * 発信プロフィールは **Notification > KakaoTalk Bizmessage > 発信プロフィール管理**タブで確認できます。
2. プッシュ通知の可否を選択します。
    * プッシュ通知を**使用しない**に選択した場合、メッセージプッシュ通知が送信されません。
3. 成人向けメッセージの可否を選択します。
    * 成人向けメッセージの場合、年齢認証前まではチャットルームメッセージにカバーがかけられ、20歳以上のみ内容を確認できます。
4. メッセージタイプを選択します。
    * テキスト
        * 韓国語/英語区分なく、スペース含めて1,300文字のテキスト+リンクボタン最大5個（縦配列）
    * 画像
        * 韓国語/英語区分なく、スペース含めて1,300文字のテキスト+画像1枚+リンクボタン最大5個（縦配列）
    * ワイド画像
        * 韓国語/英語区分なく、スペース含めて76文字のテキスト+画像1枚+リンクボタン最大2個
    * ワイドアイテムリスト
        * 1つのタイトルに3〜4個のリスト（画像+アイテム）を追加できる広告タイプの商品です。
        * 韓国語/英語区分なく、スペース含めてテキスト最初のアイテムタイトル25文字、2〜4番目のアイテムタイトル30文字+画像アイテム3〜4個+リンクボタン最大2個（横配列）
    * カルーセルフィード
        * 最大10個の画像とさまざまなテキスト情報を盛り込める広告タイプの商品です。
        * 韓国語/英語区分なく、スペース含めてタイトル20文字のテキスト+文句180文字のテキスト+画像+リンクボタン2個（横配列）で構成されたアイテム最大6個
    * プレミアム動画
        * 添付した動画が吹き出しに自動再生されるタイプです。
        * 動画リンクは、カカオTV にアップロードされた映像のみ使用できます（例：[https://tv.kakao.com/v/#{숫자}](https://tv.kakao.com/v/#%7B%EC%88%AB%EC%9E%90%7D) / [https://tv.kakao.com/channel/#{숫자}/cliplink/#{숫자}](https://tv.kakao.com/channel/#%7B%EC%88%AB%EC%9E%90%7D/cliplink/#%7B%EC%88%AB%EC%9E%90%7D)）。
        * 韓国語/英語区分なく、スペース含めてヘッダー20文字のテキスト+文句76文字のテキスト+カカオTV にアップロードされた映像1個+リンクボタン1個
    * コマース
        * 商品の価格および割引情報を強調して表示できる吹き出しです。
        * 韓国語/英語区分なく、スペース含めてタイトル20文字のテキスト+付加情報34文字のテキスト+リンクボタン最大2個（横配列）
    * カルーセルコマース
        * さまざまな商品の情報をカタログ形式で構成できる吹き出しです。
        * 韓国語/英語区分なく、スペース含めてタイトル30文字のテキスト+付加情報34文字のテキスト+リンクボタン最大2個（横配列）で構成されたアイテム最大6個
        * カルーセルコマースに使用されるすべての画像は同じ比率である必要があります。
5. 画像がある場合は画像を選択します。
    * メッセージに画像を添付するには、まず**画像管理**タブで画像を登録する必要があります。
    * 画像リンク：画像クリック時に接続されるリンクを入力します（http://、https://を含む URL）。
    * カルーセルコマースに使用されるすべての画像は同じ比率である必要があります。
6. ウェブリンク、アプリリンク、ボットキーワード、メッセージ転送、チャンネル追加、相談トーク転換、ボット転換、ビジネスフォームボタンを挿入できます。
    * 基本型最大5個、カルーセル/ワイドアイテムリスト最大2個
7. メッセージでクーポンを強調する必要がある場合、ボタンを追加してクリック時に添付したクーポンに移動するように設定できます。
8. 代替送信メッセージ設定の可否を選択します。
    * **080受信拒否管理**タブで代替送信の可否を**使用**に設定し、SMS サービスを連動する必要があります。
9. 受信者を追加します。
    * 受信者は携帯電話番号形式で作成できます。
10. 受信者別にターゲティングタイプを選択します。
    * ブランドメッセージは、広告主のマーケティング受信同意（以下「マ受同」）ユーザーおよびチャンネルフレンドユーザーを対象に送信できる広告性メッセージです。
    * ターゲティングタイプによって、指定した受信者のメッセージ受信可否が変わる場合があります。
        * M：広告主マ受同ユーザー（カカオトーク受信同意）
            * 広告主のマ受同ユーザー（カカオトーク受信同意）を対象に広告性メッセージを送信します。
        * N：広告主マ受同ユーザー（カカオトーク受信同意）- チャンネルフレンド
            * 広告主のマ受同ユーザー（カカオトーク受信同意）からチャンネルフレンドを除いて広告性メッセージを送信します。
        * I：広告主送信リクエスト対象 ∩ チャンネルフレンド
            * 広告主送信リクエスト対象で、チャンネルフレンドの場合にのみ広告性メッセージを送信します。
11. 入力を完了した後、**送信**をクリックして送信します。

<a id="mass-delivery"></a>

### 一括送信

Excel/CSV 形式のテンプレートファイルを使用して、複数の受信番号にブランドメッセージを送信できる機能です。下部のタブで**一括送信**を選択します。

![friendtalkupgrade_mass01.png](https://static.toastoven.net/prod_alimtalk/friendtalkupgrade_mass01.png)

* 内容にテンプレート置換子を「#{名前}」形式で入力した後、**テンプレートダウンロード**ボタンをクリックすると、テンプレート置換子が含まれた CSV、XLSX ファイルをダウンロードできます。
* CSV ファイルは Excel で開いて保存するとき、韓国語が正しく保存されない場合があります。
* CSV、XLSX ファイルのみアップロードできます。サイズは最大30MB で、送信できる最大受信者数は1,000,000人です。

![friendtalkupgrade_mass02.png](https://static.toastoven.net/prod_alimtalk/friendtalkupgrade_mass02.png)

* 各受信者別に個別ターゲティング値とテンプレート置換子の値を入力できます。
* テンプレート置換子なしで内容を入力すると、同じ内容ですべての受信者に送信されます。

**送信**ボタンをクリックする際、**検収後進行**または**即時送信**を選択できます。

* 検収後進行：**一括送信照会**タブで7日以内にメッセージ受信者確認後、お知らせトークを送信します。（予約送信時はサポートしません。）
* 即時送信：お知らせトーク受信者を確認せずに即座に送信します。送信状態は**一括送信照会**タブで確認できます。

<a id="fallback"></a>

### 代替送信
ブランドメッセージ送信に失敗した場合、該当内容を文字メッセージで代替送信できる機能です。
**共通内容で送信**または**ユーザー別異なるメッセージ設定**を選択して送信できます。

![friendtalkupgrade_06_20250616.png](https://static.toastoven.net/prod_alimtalk/friendtalkupgrade/friendtalkupgrade_06_20250729.png)

* **080受信拒否管理**タブで代替送信の可否を**使用**に設定し、SMS サービスを連動する必要があります。
* 代替メッセージを送信するには発信番号が必要です。代替送信の可否や発信番号を設定しない場合、**代替送信管理**タブで指定したデフォルト設定が適用されます。
* カカオトーク送信結果が失敗に変更された後に代替送信され、文字受信まで最大数十秒かかる場合があります。
* メッセージによって、ボタン、リンクなど一部内容がカカオトークメッセージと異なって表示される場合があります。
* メッセージ長に応じて SMS/LMS で代替送信されます（各タイプ別文字送信時に別途利用料金が発生）。

<a id="set-as-common-content"></a>

#### 共通内容で送信

![friendtalkupgrade_resend01_20250729.png](https://static.toastoven.net/prod_alimtalk/friendtalkupgrade/friendtalkupgrade_resend01_20250729.png)

* **発信プロフィール**を選択した後、**共通内容で送信**をクリックして代替送信メッセージを設定します。
* 代替送信メッセージを入力しない場合、[メッセージ本文] で送信されます。

<a id="set-by-different-messages-per-user"></a>

#### ユーザー別異なるメッセージ設定
* **発信プロフィール**を選択した後、**ユーザー別異なるメッセージ設定**をクリックして下部の**受信者追加**タブで代替送信メッセージを設定します。
* 追加された受信者番号横の [ペンアイコン] をクリックすると、設定された内容を確認・修正できます。

!!! danger "広告性メッセージ送信時の注意事項"
    広告性メッセージとは、送信者が経済的利益を得ることを目的として送信する情報や財貨、サービスに関するメッセージです。

    **[広告性メッセージ送信ガイド]**
    1. 広告表示
        ① 広告性内容が含まれたメッセージの最初に（広告）を表示する必要があります。
        ② 広告性メッセージに対する判断基準
            a. 特価/割引商品案内
            b. 商品およびサービス広報のためのプロモーションまたはイベント
            c. 情報を「主」に表示しても [上記 a、b] の内容が混在する場合
    2. 送信者の連絡先表示義務事項
       ① 送信者名称、送信者連絡先は、メッセージ本文より上段に作成される必要があります。
       ② 送信者名称と送信者連絡先の場合、チャットルーム上段に露出されます。
       ③ 連絡先の場合、電話番号または住所のうち一つを選択して記入する必要があります。
    3. 受信拒否意思表示を容易にできる措置および方法表示義務事項
       ① メッセージに受信の拒否および受信同意撤回の意思表示を容易にできる措置および方法を広告本文に表記して具体的に明らかにする必要があります。
       ② 同措置および方法で受信の拒否または受信同意撤回が容易に行われないか不可能な場合には、これを表記しなかったものと見なされます。
       ③ メールを受信拒否するためにウェブサイトにログインするなど、別途の追加的な措置を経る必要があるものは、受信の拒否または受信同意撤回を困難にするものとして法違反に該当します。

### 送信結果の照会

![friendtalkupgrade_07_20250616.png](https://static.toastoven.net/prod_alimtalk/friendtalkupgrade/friendtalkupgrade_07_20250729.png)

<a id="cancel-send"></a>

### 送信キャンセル
一般送信のうち、送信リクエスト日時を未来に設定した予約送信の場合はキャンセルが可能です。

* 予約送信リクエストを照会すると、リクエストIDの左側にチェックボックスを確認できます。
* チェックボックスは、キャンセルされていない予約リクエストにのみ表示されます。
* キャンセルしたいリクエストのチェックボックスを選択した後、上部の**選択予約キャンセル**ボタンをクリックすると、該当リクエストがキャンセルされます。
* 照会一覧のヘッダにあるチェックボックスで、該当一覧の全体選択及びキャンセルが可能です。

## 画像管理

![friendtalkupgrade_8_20250616.png](https://static.toastoven.net/prod_alimtalk/friendtalkupgrade/friendtalkupgrade_8_20250616.png)

ブランドメッセージに使用する画像を登録または削除し、登録された画像の情報を確認できます。

* 画像を（基本/ワイド、ワイドアイテムリストタイプ、カルーセルフィード、カルーセルコマース、ビジネスフォーム）種類別に選択して登録できます。
* 画像削除時、画像を使用できなくなります。
* テンプレート修正時に画像を他の画像に変更すると、既存画像がカカオ CDN から削除され、URL が無効になります。同じ画像を使用する他のテンプレートも影響を受けるため注意が必要です。画像照会 API やコンソールでは画像情報が維持されますが、実際の画像にはアクセスできないため、元のファイルは自社サーバーに別途保管することをお勧めします。
* ファイル規格、推奨/制限サイズを必ず遵守します。
* 画像アドレス URL をコピーできます。該当 URL は API 送信時に活用されます。
* ビジネスフォームはカカオトークユーザーが手軽に参加できる予約、アンケート、応募などのイベント設計をサポートするビジネスツールで、ビジネスフォームを作成して該当 ID を入力します。
* [\[ビジネスフォーム登録はこちら\]](https://business.kakao.com/talkbizform/)
* [\[ビジネスフォーム登録ガイドはこちら\]](https://kakaobusiness.gitbook.io/main/tool/bizform)

<a id="image-upload-allowance"></a>

#### 画像アップロード許可範囲

* ファイル形式: JPG、PNG
* 各画像アップロード API 仕様を確認してください。

## 動画管理

![friendtalkupgrade_video_management.png](TODO: 콘솔 캡처 후 NHN static 호스트 URL 교체)

ブランドメッセージに使用する動画を登録または削除し、登録された動画の情報を確認できます。

* 発信プロフィールを選択した後、動画を登録または削除できます。
* 動画アップロードは、コンソール画面でファイル選択後にアップロードボタンを押すと、進捗状況が表示されます。アップロードが進行中の間は、ページを更新したり閉じたりしないでください。進行中に更新が発生すると、アップロードが中断され、再度アップロードする必要があります。
* 登録した動画は、カカオビズセンターでエンコードが完了された後、送信に使用できます。エンコードは通常 5〜10 分かかり、動画の長さによって最大 3 日までかかる場合があります。3 日を超えると、自動的に `ERROR` 状態として処理されます。
* 動画ステータスは、カカオビズセンターと定期的に同期されます。`PUBLIC` 状態の動画のみテンプレート登録および送信が可能で、`PRIVATE` 状態の動画はテンプレート登録のみ可能です。
* 登録された動画は、カカオ側で永久保存され、コンソールで動画を削除してもカカオビズセンターの動画は自動的に削除されません。カカオチャンネル管理者がチャンネルビジネスホームの管理画面で直接削除できます。
* 動画削除時に該当動画が使用されたテンプレートからは送信できなくなるため、注意が必要です。

<a id="video-upload-allowance"></a>

#### 動画アップロード許可範囲

* ファイル形式：MP4、MOV、AVI
* 最大ファイルサイズ：4GB
* 最大動画長：4 時間
* 最大解像度：8K

## テンプレート管理

![friendtalkupgrade_09_20250616.png](https://static.toastoven.net/prod_alimtalk/friendtalkupgrade/friendtalkupgrade_09_20250729.png)

* 従来の「お知らせトーク」と同様の方法でテンプレートを管理し、送信に使用できます。
* ブランドメッセージには検収プロセスがなく、作成、修正、削除が自由に行えます。
* 「お知らせトーク」とは異なり、テンプレートコードをユーザーが登録する方式ではなく、カカオからランダムな識別子を付与される形式です。

<a id="080-opt-out-management"></a>

## 080 受信拒否管理

* ブランドメッセージでは、NHN Cloud の SMS サービス連携により「080 受信拒否管理」と「代替送信」機能を 1 つの smsAppkey で統合管理します。

<a id="register-and-manage-080-opt-out-numbers"></a>

### 080 受信拒否番号登録および管理
* ブランドメッセージの手動ユーザー送信の場合、発信プロフィールでフレンドになっていない受信者にメッセージを送信できるため、発信プロフィールに 080 受信拒否番号を登録する必要があります。
    * 発信プロフィールの 080 受信拒否番号は、同一トークチャンネルの他の組織、他のプロジェクト、他のディーラーの発信プロフィール全体に一括適用されるデータのため、慎重に変更する必要があります。
* 発信プロフィールに 080 受信拒否番号を登録または修正が必要な場合は、**NHN Cloud カスタマーサポート > お問い合わせ**にお問い合わせください。
* 080 受信拒否番号を登録した場合、発信プロフィールの所有者には該当番号で受信拒否の意思を表示した受信者にメッセージが送信されないよう管理する責任が与えられます。
* NHN Cloud SMS サービスと連携すると、080 受信拒否番号の受信拒否管理を簡単に行うことができます。
  ![sms 080](https://static.toastoven.net/prod_sms/SMS_14_20230818.png)
* SMS サービスで 080 受信拒否番号を管理するには、まず SMS サービスを有効にする必要があります。
* 管理したい 080 受信拒否番号を SMS サービスに登録します。
    * もし発信プロフィールに登録された 080 受信拒否番号が SMS サービスに登録されていない場合、送信リクエストはすべて失敗します。
* 詳細については、[SMS ユーザーガイド](https://docs.nhncloud.com/ko/Notification/SMS/ko/console-guide/#080)を参照してください。
  ![friendtalkupgrade_10_20250616.png](https://static.toastoven.net/prod_alimtalk/friendtalkupgrade/friendtalkupgrade_10_20250729.png)
* 080 受信拒否番号を登録した SMS サービスのアプリキーを 080 受信拒否タブに入力し、保存します。
* 管理対象として指定した SMS サービスで、発信プロフィールに登録されている 080 受信拒否番号で受信者管理が可能かどうかは、コンソールの **080 受信拒否管理**タブで該当発信プロフィールの **SMS プロジェクト 080 番号管理可否**項目で確認できます。
* SMS サービスアプリキーを空にして保存した場合
    * お客様が直接受信拒否受信者を管理する必要があり、ブランドメッセージ送信過程で受信者に対する検証は行いません。

<a id="manage-fallback"></a>

### 代替送信管理
* ブランドメッセージの送信に失敗した場合、該当内容を文字メッセージで代替送信するよう設定できます。
* NHN Cloud SMS サービスを利用している必要があり、メッセージの長さに応じて SMS/LMS で送信されます（各タイプ別の文字送信時に別途利用料が発生）。
* SMS アプリキーを修正すると、すべての検索用 ID のチャンネルは代替送信設定が初期化されます。
* SMS サービスで代替送信されるため、SMS サービスの送信 API 仕様に従ってフィールドを入力する必要があります（SMS サービスに登録された発信番号、各種フィールド長の制限など）。
* 指定した代替送信タイプのバイト制限を超える代替送信件名や内容は、切り詰められて代替送信される場合があります（[SMS API ガイド > 注意事項](https://docs.nhncloud.com/ko/Notification/SMS/ko/api-guide/#_1) を参照）。
* 代替送信内容は EUC-KR 基準で送信され、サポートしていない絵文字は代替送信に失敗します。
* 代替送信設定を修正する検索用 ID のチャンネルは、**修正**をクリックして **代替送信可否**および **発信番号**を修正できます。
* **代替送信可否**を **使用しない**に設定した場合、ブランドメッセージ送信時に該当検索用 ID のチャンネルは代替送信できません。
* ブランドメッセージの広告メッセージは広告 SMS API で代替送信されるため、必ず 080 受信拒否番号を登録してから代替送信されます。
* ブランドメッセージの広告メッセージの resendContent フィールドを入力する場合、SMS 広告 API の広告文句を必須で入力してから代替送信されます。`(広告)内容[無料受信拒否]080XXXXXXX`
* ブランドメッセージの広告メッセージの resendContent フィールドがない場合、登録された 080 受信拒否番号で広告文句を自動生成して代替送信されます。

## 非友だちメッセージ送信(ターゲティング M, N)の利用申請

* 非友だちメッセージ送信(ターゲティング M, N)の利用をご希望の場合は、利用申請を行う必要があります。利用申請をしない場合、送信時にM/Nタイプが表示されません。
* 利用申請は、以下の条件を満たす場合に承認されます。
    * ビジネス認証チャネル
    *  事業者番号の登録
    *  チャネルのカスタマーセンター電話番号の登録
    *  チャネルの友だち数が5万人以上
    *  3か月以内の「お知らせトーク」送信成功履歴を保有

<a id="cautions"></a>

### 注意事項
* ビジネス認証が解除された場合、非友だちメッセージ送信(ターゲティング M, N)の利用権限が解除されます。ビジネス認証の再審査後に、再度利用申請を行う必要があります。
* 広告性情報の受信同意に関する証跡資料ファイルはトークチャネル単位で保存されるため、変更すると同一トークチャネルの他の販売代理店の送信プロフィールに一括で適用されます。
* 他の販売代理店がアップロードしたファイルが既に存在する場合、ファイルアップロードのプロセスを省略して非友だちメッセージ送信(ターゲティング M, N)の利用申請が可能です。
